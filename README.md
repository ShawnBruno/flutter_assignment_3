Create a Flutter App 

1. Splash Screen UI
    Design a splash screen with the following:
    a. App logo in the center
    b. Progress indicator OR tagline text
    c. Auto-navigate after 2–3 seconds to the next screen

2. Add Theme and apply it throughout the app.

3. Login Screen UI
    Create a Login screen with:
    a. Email field
    b. Password field
    c. Login button
    d. "Forgot password?" text
    e. Simple validation messages
        - Email cannot be empty
        - Password cannot be empty
        - valid email format check

4. Sign Up Screen UI
    Create a Sign Up screen with:
    a. Name field
    b. Email field
    c. Password field
    d. Confirm password field
    e. Register button
    f. A link/text button: “Already have an account? Login”
    g. Add basic validation:
        - All fields required
        - Password and confirm password should match

 5. Add Onboarding Screen (1-2 page only)
     a. Illustration image
     b. 2–3 lines text
     c. Next button

Suggestions:
1.  Use SizedBox, Center, Row, Column, Image, Text, Form, TextFormField, ElevatedButton, TextButton, SingleChildScrollView (if required), CircularProgressIndicator etc
2. SmoothPageIndicator (for onboarding)
